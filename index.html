Few Words About Us

We are a team of developers who are strong believers of internet privacy and freedom, which is why we are making software to keep you safe, and secure while online.

All Packaged

With Eglo Cloud, you have the option to install the same software that we run on our servers, on your own server, which is why we include a simple "install.php" file that you just need to run to get everything all setup.

Versioning

We get it, you accidentally edit a file and refresh the webpage, no more "Ctrl + Z". With Eglo Cloud we have file versioning, which makes restoring your old edits easy and fast. Eglo Versioning is also available on your personal server too.

Secure

We don't want anyone accessing your account without your permission, so we keep an eye out for any suspicious activity, including: New IP address, New geographic location, and sudden account activity after being inactive for a long period of time.
